## Start Server Locally

```
npm i -g pnpm

pnpm i
pnpm run dev
```

Và sau đó truy cập `http://localhost:3000`

Hoặc cài đặt [Vite extension cho VS Code](https://marketplace.visualstudio.com/items?itemName=antfu.vite) để chỉnh sửa song song.
